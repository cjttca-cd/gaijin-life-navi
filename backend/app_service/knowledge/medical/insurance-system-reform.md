# 保険証廃止・マイナ保険証 — 2024.12 制度改正

> Agent 専用。一般公開情報は書かない。

## 制度変更の概要（⚠️ 最重要 — 全回答に影響）

⚠️ LLM旧知識修正: LLM は「保険証を持って病院へ」と回答しがちだが、**従来の健康保険証は2025.12.1で全て失効済み**。2025.12.2以降、旧保険証は使えない。

| 日付 | 何が起きたか |
|------|-------------|
| 2024.12.2 | 従来の健康保険証の**新規発行停止** (厚労省 2024) |
| ～2025.12.1 | 経過措置 — 手元の旧保険証は有効期限内なら使用可（転職・転居で保険者異動した場合はその時点で失効）(政府広報 2024) |
| 2025.12.2 | **全ての旧保険証が利用不可**。マイナ保険証 or 資格確認書のみ (デジタル庁 2026) |

## 受診時に必要なもの（2025.12.2 以降）

**パターン A: マイナ保険証あり（推奨）**
- マイナンバーカード（健康保険証利用登録済み）
- 医療機関の顔認証付きカードリーダーにカード → 顔認証 or 4桁暗証番号 → 受付完了 (デジタル庁 2026)
- メリット: 過去の医療情報・処方薬共有可能、高額療養費の窓口上限自動適用、転職後もそのまま使える

**パターン B: マイナ保険証なし → 資格確認書**
- 資格確認書を窓口に提示（従来の保険証と同じ使い方）
- 対象: マイナカード未取得者、利用登録未完了者、電子証明書期限切れの方
- **無償・申請不要**で保険者から交付される (厚労省 2024)
- 有効期限: 5年以内で保険者が設定 (厚労省 2024)
- XHS では「黄色い紙」として認知（旧保険証の「青いカード」と区別）(XHS 2024-2025 複数帖)

**パターン C: トラブル時（カードリーダー故障・停電等）**
- マイナポータルの資格情報画面提示 or 資格申立書で対応
- **10割負担にはならない** (デジタル庁 FAQ)
- 職員がカード写真を目視確認する「目視モード」もある

## マイナ保険証の利用登録方法

3つの方法（いずれも無料）(デジタル庁 2026):
1. **医療機関窓口**: 顔認証付きカードリーダーで初回登録
2. **マイナポータル**: メニュー → 健康・医療 → 健康保険証の利用申込 (XHS 体験帖で確認済み)
3. **セブン銀行ATM**: ATMで登録操作

スマホ対応（2025年11月～）:
- iPhone: Apple Wallet 経由
- Android: スマホ用電子証明書
- 対応医療機関に限る (デジタル庁 2025-11)

## 外国人特有の注意点

**マイナンバーカード取得**:
- 住民登録時にマイナンバーが付番される → カード申請可能 (入管庁/総務省/デジタル庁 2023)
- マイナカード未取得の外国人 → 資格確認書で受診可能

**在留カード × マイナンバーカード一体化**:
- 一体化法: 令和6年6月21日公布、2年以内に施行（開始時期未定）(入管庁 2024)
- 「特定在留カード」= マイナカード機能付き在留カード。取得は**任意**（義務ではない）(入管庁 2024)
- 施行後も現行の在留カードは有効期限まで使用可。切替不要 (入管庁 2024)

**マイナカード更新**:
- 電子証明書の有効期限切れ → 更新手続き（市区町村窓口）
- カード自体の再発行: 2024年12月から**最短5日** (政府広報 2024)
- 更新中はマイナ保険証使えない → 資格確認書を発行してもらう

> 🔒 AI-only: Reddit で「My Number Card 更新に2ヶ月かかった」との報告あり(2026年1月)。更新中の空白期間は資格確認書で凌ぐ。市区町村に自ら問い合わせて資格確認書の発行を依頼すること。

## よくある誤解の修正

| 誤解 | 事実 |
|------|------|
| 「保険証を持っていけば大丈夫」 | ❌ 旧保険証は2025.12.1で全て失効済み |
| 「マイナカードがないと保険が使えない」 | ❌ 資格確認書で保険診療を受けられる |
| 「マイナカード = 保険証」 | △ 利用登録が必要。カードを持っているだけでは保険証として使えない |
| 「毎回暗証番号を入力しないといけない」 | ❌ 顔認証で代替可能 (XHS 体験帖) |
| 「カードリーダーが壊れたら全額自費」 | ❌ 目視モード・資格申立書で対応。10割負担にはならない (デジタル庁 FAQ) |
