# 永住申請の実態

> Agent 専用。一般公開情報は書かない。

## 税金・社会保険: 最大の落とし穴

- **住民税1回でも遅延 = 不許可**: 修士卒・一流企業・80点以上・教授の推薦状つきでも、2ヶ月遅延1回で不許可 (Reddit: 190 votes)
- **遡及納付は救済にならない**: 未納分を後から全額払っても「遅延した」事実は消えない
- **「完璧な2年間」が必要**: 直近2年間に1日の遅延もないことが実質要件
- **普通徴収は罠**: 転職で特別徴収→普通徴収に切り替わるとコンビニ払い忘れリスク急増 → 転職後すぐに新会社で特別徴収に切替
- **年金**: 2年分完納必須。口座振替を設定すべき

## 収入の罠

- **住宅手当は年収に含まれない**: 福利厚生は課税対象外 → ポイント計算の年収に含められない (Reddit)
- **収入証明は課税証明書の金額が基準**: 会社の契約書や給与明細ではない

## 在留期間の罠

- **1年ビザでの永住申請はほぼ不許可**: 3年以上が事実上の必須要件
- 1年ビザが続く → 永住申請不可 → 永住なしで再び1年更新、の悪循環

## 審査期間の実態

- **公式 4ヶ月 → 実態 8〜12ヶ月**: 品川は特に遅い (Reddit: 10ヶ月待ち)
- **地方入管は圧倒的に早い**: 関東以外に住んでいることが審査スピード上の大きなアドバンテージ
- 追加資料請求が1〜3回来るのが普通 → そのたびに1〜2ヶ月加算

## 拒否理由の聞き方

- 不許可通知は定型文のみ（具体的理由なし）
- **入管に出頭して口頭で聞くしかない**: 通知受領から20日以降
- 審査官は情報を持っているが**申請者には見せない**
- 日本語に不安がある場合は日本人の同席を推奨

## 高度専門職（HSP）永住の罠

- 80点 = 1年で申請可能は制度上の話。実態:
  - ポイントは**申請時点と過去の時点**の両方で基準を満たす必要あり
  - 70点: 過去3年間ずっと70点以上の証明が必要
  - 80点: 過去1年間ずっと80点以上の証明が必要
- **遅延納付後の再申請待機**: 70点 = 3年、80点 = 1年
- 技人国のままでもポイント計算で永住申請可能（HSPビザへの変更不要）

## 理由書の書き方

- 自由書式だが**法的要件3点に対応する構成**にすべき:
  1. 素行が善良 2. 独立した生計 3. 日本国の利益に合すること
- 家の購入、永住の意思等の具体的コミットメントが有効
- 推薦状（教授・上司）は加点要素

## 永住取得後の盲点

- **在留カードの更新忘れ**: 永住者でも7年の有効期限あり → 更新忘れ = 携帯義務違反
- **1年以上の出国で永住喪失**: みなし再入国許可の期限超過 → 再入国許可（最長5年）を事前取得
- **税金・社会保険の未納が永住取消事由に追加（2022年改正）**: 取得後も滞納は致命的

## 居住実態チェック

- 申請前1年間に**9ヶ月以上日本に居住**していないと不許可リスク → 出張多い人は出入国記録をカウント

## 行政書士を使うべきか

- **不要**: 書類が揃っていて日本語ができる人。自分で結果は変わらない (Reddit)
- **必要**: 不許可歴あり / 複雑な職歴 / 税金遅延歴 / 日本語不安
- 詳細は `immigration-tips.md` の行政書士セクション参照
