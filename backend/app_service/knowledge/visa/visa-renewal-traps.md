# 在留期間更新の落とし穴

> Agent 専用。一般公開情報は書かない。

## 更新は「当然」ではない

- 10年以上在日でも技人国更新不許可の実例あり (XHS) → 長期在留 ≠ 自動更新
- 経営管理ビザの更新審査が年々厳格化 (XHS: 複数投稿)
- 「拒否増加トレンド」の体感報告が急増 (XHS: 「外国人迎来冰河时期」850 likes)

## 在留期間の格下げ（5年→3年→1年）

- 更新時に5年→3年に格下げされるケースあり (XHS: 111 likes)
- **付与年数は実質ランダムに見える**: 同じ会社で同時申請、N2持ちが1年、日本語ゼロの同僚が3年 (Reddit)
- 格下げの主な要因（推測）: 転職直後 / 有期契約 / 収入低下 / 税金・社保遅延
- **1年が続くと永住申請要件（3年以上）を満たせない連鎖リスク**

## 書類不備の罠

- 1つでも足りないと**即日帰宅** → 再訪問
- **在留カードの氏名表記と会社書類の不一致**（繁体字 vs 簡体字、カタカナ）で拒否される実例 (知乎)
- 追加資料を9点請求された後、2週間で許可の事例あり (XHS) → 追加資料 ≠ 不許可フラグ

## タイミングの罠

- 3ヶ月前から申請可能だが**ギリギリは危険** → 追加資料・繁忙期遅延で間に合わなくなる
- **特例期間中の出国**: 結果通知を受け取れないリスク → 審査中は出国しない方が安全
- 繁忙期は処理期間が倍以上: 3/13申請→6/21通知の実例 (Reddit)

## 転職時の更新リスク

- 転職後3ヶ月以上無職 → 不許可リスク急増
- 転職直後の更新 → 1年が多い傾向
- **就労資格証明書**を転職時に取得 → 次の更新がスムーズ（詳細は `status-change.md` 参照）

## 通知はがきの読み方

- **はがき = 基本的に良いニュース**（結果受取の案内）
- **書留 = 追加資料請求 or 不許可通知**の可能性が高い
- 不許可理由は書面に書かれない → 入管に出頭して口頭で聞く（詳細は `permanent-residency.md` 参照）
