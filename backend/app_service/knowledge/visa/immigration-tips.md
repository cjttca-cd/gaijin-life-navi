# 入管局の実態と攻略

> Agent 専用。一般公開情報は書かない。

## 品川入管（東京出入国在留管理局）

- **待ち時間: 2〜5時間が常態**: 半日がかりを覚悟。全休を申請すべき（半休では足りない）
- 到着推奨: **開館30〜60分前**（9:00開館）。**閉館30分前（15:30頃）**が穴場との報告もあり（ただしギャンブル）
- バス: 品川駅港南口バス停24番（品99系統）、開館前から行列

## 品川以外の選択肢（東京圏）

- **松戸出張所**: 東京の複数の区を管轄しており品川より大幅に空いている。管轄区域は出入国在留管理庁の公式サイト（`web_search で「出入国在留管理庁 管轄区域」を検索`）で確認 → 対象地域在住なら品川より強く推奨
- **立川出張所**: 品川より空いているが「劇的に良い」わけではない。4時間待ちの報告もあり (Reddit)
- **新百合ヶ丘出張所**: 対象の東京アドレスが限定的だが、利用できる人にとっては「15分で完了」の天国 (Reddit)
- **横浜支局**: 神奈川在住者は品川より圧倒的に良い（待ち時間が短い）
- **判断ロジック**: `web_search で「出入国在留管理庁 管轄区域」を検索` → ユーザーの住所に対応する入管局を特定 → 品川以外に選択肢があれば積極的に推奨

## 繁忙期カレンダー

- **最混雑**: 3〜4月（卒業・就職）、9〜10月（後期入学）
- **週内**: 月曜・金曜が混雑。**火〜木が狙い目**
- **在留カード受取のみ**: 予約システムで行列スキップ可能

## 予約システム（多くの人が知らない）

- 東京入管は一部手続きで**オンライン予約**が可能（英語・中国語対応）
- 在留カード受取時は特に有効 → 予約枠がある場合、行列をスキップできる
- **多くの外国人がこの制度の存在を知らない** (GaijinPot)

## 電話問い合わせの裏技

- **品川入管はほぼ繋がらない**: 終日コールしても自動音声のみの事例 (Reddit)
- **裏技**: 英語オプションではなく日本語オプション（111）を選ぶと繋がりやすい (Reddit)
- 一般情報番号（0570 013 904）→ 各部門に転送してもらう方が早い場合あり
- 「忙しい」自動応答が出ても**切らずに待機** → 待ち行列に入れることがある
- 最終手段: 直接出向く（品川入管2階P相談カウンター）

## 入管窓口での対応

- **担当官による裁量の幅が大きい**: 同じ申請でも求められる書類が異なる場合あり
- 日本語を使った方がスムーズ（英語対応は限定的）
- 書類不備は**その場で修正不可** → 帰宅して修正し再訪問
- **追加資料の請求 ≠ 不許可フラグ**: むしろ審査が進んでいる証拠

## オンライン申請の実態

- 対象: 在留期間更新、在留資格変更、再入国許可、資格外活動許可
- **PC限定**（スマホ不可）、マイナンバーカード + ICカードリーダー必須
- 全書類を1つのPDFに統合＋ファイルサイズ制限 → スキャン環境が必要
- **システムエラー頻発** (XHS)。ただし処理は窓口より早い傾向
- **在留カード受取は入管窓口** → 完全にオンラインでは完結しない

## 行政書士の使い方

- **申請取次行政書士**: 本人の代わりに入管に申請可能（本人が行かなくてよい）
- **使うべき場面**: 平日休めない / 不許可歴あり / 複雑なケース（経営管理等）/ 日本語不安
- **不要な場面**: 単純な更新（同じ会社・同じ在留資格の延長）→ 自分で結果は変わらない (Reddit)
- **会社の人事部門**: 申請取次の届出をしている企業なら代理申請可能（費用ゼロ）

## FRESC（外国人在留支援センター）

- 四ツ谷の無料相談窓口（20言語対応）
- **入管への直接相談より先にFRESC** → 必要書類の確認や手続き整理ができる

## 品川入管への持ち物

- 全書類 + **全書類のコピー**（追加で求められることがある）
- 収入印紙（局内売店は行列になる場合あり）
- **暇つぶし道具 + 充電器 + 軽食**（2〜5時間待ち）


## 留学签証申請 / COE 実務

- 財力証明開具時機の罠：签證被拒案例中、存款証明の開具時間が6か月超過（領事館要求は6か月以内）。再申請時は資金来源説明書及び担保人証明の補充が必要 (XHS 2023)
- 中国申請者の存款証明金額参考：通常は全留学期間の学費+生活費をカバー、約150万円以上が必要。銀行の捺印原件+6か月分の流水が必須 (XHS 2022-2024)
- 2023年3月17日以降、日本留学签證申請表が更新：COE原件の提出は不要で複印件のみ。新表に「在留資格認定証書番号」の記入欄が追加 (XHS 2023)
- 申請表「予定在日滞留期間」の記入テクニック：「一年三か月」「二年三か月」と書く方が日数（457日/822日）を書くより簡単で、領事館も同様に受理する (XHS 2023)
