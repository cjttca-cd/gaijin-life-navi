# 在留資格変更の罠

> Agent 専用。一般公開情報は書かない。

## 転職時の変更判断

- 同じ在留資格の範囲内（例: IT→IT、両方技人国）なら**変更不要**。ただし所属機関変更届は14日以内に必須
- 業種が大きく変わる場合 → **就労資格証明書**を取得して事前確認
- **変更申請中は前の在留資格の活動範囲のみ**: 留学→技人国の変更中は週28時間のみ就労可

## 変更申請の期間リスク

- 更新より長い（更新: 2週間〜1ヶ月、変更: 1〜3ヶ月）
- 地域差大: 品川は遅い、地方は早い。北海道は7〜10月が繁忙期（スキーリゾート関連）
- **3.5ヶ月以上かかって在留期限切れ、経済的困窮した事例あり** (Reddit)

## 観光ビザからの変更（最大の罠）

- **原則認められない**（入管の方針）
- 例外: 日本人の配偶者等（結婚）→ ただし担当官の裁量が大きい
- COEを日本国内で受け取って変更する方法はあるが、**入国前にCOE取得→観光入国→変更は拒否増加**
- 配偶者ビザの場合: 観光→変更処理に5ヶ月かかった事例あり（変更申請中は合法滞在だが就労不可）

## 留学→就労

- 卒業前に内定→変更が王道。**初回は1年ビザが多い**（3年は稀）
- 就活未定 → 「特定活動」（就職活動）への変更可能（最長1年、大学推薦状必要）

## 技人国→経営管理（起業）の罠

- **審査が最も厳しい**カテゴリの一つ
- **虚偽の事業計画 = 永久にビザ申請不利**: 造假に対する「零容忍」(知乎)
- 「見込み」だけでは不十分 → 実績ベースの証拠が必要
- 更新時にも事業実態を毎回チェック → 赤字続きは更新不許可リスク

## 配偶者ビザの注意点

- 離婚後6ヶ月以内に変更、14日以内に届出（※ 詳細は `legal` domain の family-law 参照）
- **婚姻実態チェックが厳格化**: 同居証明、コミュニケーション履歴など (XHS)
- 配偶者ビザ → 永住は最短ルートの一つ（婚姻3年+在留1年）

## 家族滞在の制限

- フルタイム就労は**在留資格変更が必要**
- 資格外活動許可で週28時間まで可能だが、**超過は本人と主たるビザ保持者の両方に影響**

## 変更中の就労と不許可時の対応

- 許可はがき受取後、**新しいカードを受け取った瞬間から**前の活動は不可
- 不許可の場合: **出国準備期間31日** → 別の在留資格を申請可能だが成功率は低い

## ビザ「翻車」3大原因（元警察官の助言, XHS）

1. 在留資格と実際の活動の不一致
2. 届出義務の不履行
3. 犯罪歴・交通違反の蓄積


## 新生児の在留資格（外国人親の子供）

### 在留資格「取得」vs「認定」：最重要な取り違え

- **日本国内で生まれた新生児に申請するのは「在留資格取得許可申請」（出生後30日以内）**。「在留資格認定証明書交付申請」は海外から日本に来る場合の手続きで別物。名称が類似しているため混同した実例があり、入管から申請の取り消し・再提出を求められたケースがある (知乎 2021)
- 海外で生まれた子供を日本に連れてくる場合は「在留資格認定証明書交付申請」（日本在住の親が入管に申請）が必要 (XHS 2022)

### 在留カード取得の現実的タイムライン

- 外国人親夫妻（NZ + AU）の実例：出生後30日以内に入管訪問 → **当日に在留カードを交付**。申請書に「護照は1年後に申請予定」と記入しても受理される（新生児護照を即日準備する必要はない）(Reddit r/japanlife 2024)

## 留学→就労変更 実務詳細

- 語学学校から直接「技術・人文知識・国際業務」への変更は学歴障壁あり：技人国は「大学卒業以上の学歴または10年の関連経験」が要件で、語学学校の学歴は算入されない。語学学校卒業後に就労签證を取得するには別ルート（特定技能等）が必要か、大学学歴が必要 (Reddit r/movingtojapan 2022)
- 留学→就労変更の実操材料清単（XHS当事者の成功例、3月初提出→3月末完了、約20日）：
  - 会社提供：①法定調書合計表 ②会社登記簿謄本 ③決算報告書 ④会社案内/パンフレット ⑤雇用契約書コピー ⑥在留資格変更許可申請書（会社用・後5頁）
  - 本人提供：①写真40×30 ②護照原本 ③在留カード ④学位修了見込書or卒業証 ⑤履歴書 ⑥在留資格変更許可申請書（申請人用・前2頁）
  - 签證費4,000円（便利店で収入印紙購入）。申請通過後に葉書で通知、葉書+護照+在留カードで受領 (XHS 2024)
- 材料準備前に必ず電話0570-013-904（東京入管局材料部門）で個人の追加材料を確認すること。各人の状況で追加材料が異なり、確認なしでは無駄足になる (XHS 2024)
- 税務記録との関連：留学から就労への変更時、入管局は直接税務を核査しないが、就職先が税務証明を要求する場合がある。打工期間中に未申告の場合、変更前に主動的に確定申告の補報を推奨 (Reddit r/japanlife 2022)

## 特定活動（就職活動継続）

- 大学/短大/大学院/専修学校卒業後に申請可能。毎回6か月、延長1回で合計最長12か月 (visa-station.jp/fujii-gyousei.com 2024)
- 重要な資格条件：**在校期間から継続して就活していること**が必須。中途停止後の再開は続签条件を満たさない。実質的な就活記録（面接招待メール、申請会社リスト、就職指導参加証明）が必要 (dsg.or.jp 2024)

## 退学・休学後の資格外活動

- 退学/休学後の打工資格は**2週間以内に自動失効**。在留カードが有効期限内でも直ちに就労を停止し、在留状態の対策（出国または在留資格変更）を検討する必要がある。在校時に合法的に打工していた人の多くがこの点を知らない (jiemo.net 2023)
