# 留学生・会社員・自営業者 — 税務フロー比較

> Agent 専用。一般公開情報は書かない。

## 留学生の税務

### 租税条約免税（⚠️ 国ごとに全く違う — 一般化禁止）

⚠️ LLM旧知識修正: LLM は「留学生のアルバイト代は一般的に免税」と答えがちだが、**国によって全く異なる**。中国は全額免税だがインド・ベトナムはアルバイト代免税なし。

| 国籍 | 免税範囲 | 条件 |
|------|---------|------|
| **中国** | アルバイト代含む全額免税 | 教育目的で滞在。金額上限・期間制限なし (国税庁 質疑応答事例 2025) |
| **韓国** | 免税あり | 滞在期間・金額に制限あり (attax.co.jp 2023) |
| **タイ** | 免税あり | 滞在期間等に制限あり (国税庁 2025) |
| **インドネシア** | 免税あり | 滞在期間・金額に制限あり (国税庁 2025) |
| **インド** | 国外送金の学資のみ | **アルバイト代は免税対象外** (国税庁 2025) |
| **ベトナム** | 国外送金の学資のみ | **アルバイト代は免税対象外** (国税庁 2025) |
| **マレーシア** | 国外送金の学資のみ | **アルバイト代は免税対象外** (国税庁 2025) |

⚠️ **改正動向**: 日中租税条約の学生免税規定について、政府が撤廃を検討中（国際標準に合わせる方針）。米国・シンガポール・マレーシアとの条約改正時には既に削除済み (産経新聞 2023年6月; attax.co.jp 2023)

**手続き方法** (国税庁 A3-11):
1. 「租税条約に関する届出書」を作成（正副2部）
2. アルバイト先（支払者）に提出
3. 支払者が所轄税務署長に提出
4. **提出期限**: 最初の給与支払日の前日まで
5. **添付書類**: 在学証明書

### 103万の壁 → 123万の壁（令和7年改正）

⚠️ LLM旧知識修正: 令和7年分から壁が変更。LLM が「103万円」と答えたら古い。

| | 令和6年分まで | 令和7年分から |
|---|---|---|
| 基礎控除 | 48万円 | **58万円** |
| 給与所得控除 | 55万円 | **65万円** |
| 所得税ゼロの壁 | **103万円** | **123万円** |
| 配偶者控除満額 | 配偶者年収103万以下 | 配偶者年収**123万以下** |
| 配偶者特別控除満額 | 103万～150万 | 123万～**160万** |

(国税庁 令和7年度税制改正; XHS「Mayの在日干货笔記」2025)

### 勤労学生控除

⚠️ LLM旧知識修正: 旧基準は「合計所得65万円以下」→ 令和2年分以降は**75万円以下**、さらに令和7年分からは**85万円以下**。

**条件（3つすべて）** (国税庁 No.1175, 令和7年4月1日法令):
1. 勤労による所得があること
2. 合計所得金額が75万円以下（令和7年分から85万円以下）、かつ勤労以外の所得10万円以下
3. 学校教育法に規定する学校の学生・生徒であること

**控除額**: 27万円
**申請**: 「扶養控除等（異動）申告書」に記載して勤務先に提出、または確定申告書に記載

### 留学生の最重要ルール: 収入ゼロでも住民税申告は必須

- **国保の減免判定に使われる** → 申告しないと国保が大幅に上がる (XHS 2025 複数帖で頻出)
- 確定申告した場合は住民税の別途申告は不要
- 打工先が年末調整済み + 他に収入なし → 確定申告不要（情報は自動で市区町村に行く）
- **収入ゼロの場合**: 市区町村に住民税申告書を提出（収入欄0で提出）

(XHS「毛豆豆LY」2025/02 357👍; XHS「文龙学长」2025/02 207👍)

### 留学生 年間フロー

| 時期 | やること |
|---|---|
| バイト開始前 | 租税条約の適用可否確認（国籍による） → 適用可能なら届出書をバイト先経由で税務署に提出 |
| 毎月 | バイト先が源泉徴収（租税条約適用者は免除） |
| 12月 | バイト先で年末調整（1ヶ所のみの場合）。源泉徴収票を受領 |
| 翌年1-2月 | 区役所から住民税申告書が届く |
| 翌年2/16〜3/15 | 確定申告（2ヶ所以上バイト or 年末調整されてない場合） |
| 翌年3月頃 | **住民税申告（収入0でも必ず！）** |
| 卒業時 | 就職 → 新会社に源泉徴収票提出 / 帰国 → 準確定申告+納税管理人届出 |

## 会社員の税務

### 確定申告の要否判断

**不要（年末調整で完結）**:
- 1ヶ所から給与 + 年末調整済み + 副業なし
- 住宅ローン控除2年目以降、生命保険料控除等 → 年末調整で処理可

**必須** (国税庁 No.1900, 令和7年法令):
1. 年収 2,000万円超
2. 副業所得（給与・退職を除く）20万円超
3. 2ヶ所以上から給与（条件付き）
4. 源泉徴収義務のない者からの給与

**任意だが得**:
- 医療費控除、ふるさと納税（5ヶ所超）、住宅ローン控除1年目

### 副業20万円ルールの落とし穴

| | 所得税 | 住民税 |
|---|---|---|
| 20万円以下 | 確定申告不要（特例） | **申告必要（特例なし）** |
| 20万円超 | 確定申告必要 | 確定申告すれば自動反映 |

⚠️ 「20万円以下なら何もしなくていい」は誤り。**住民税は1円でも申告義務あり**。確定申告しない場合は市区町村に住民税の申告書を別途提出 (freee 2026; sogyotecho.jp 2025)

⚠️ 医療費控除やふるさと納税で確定申告する場合 → 20万円以下の副業所得も含めて全て申告が必要 (freee 2026)

### 転職年の処理

- 転職先に前職の源泉徴収票を提出 → 合算年末調整で完結
- 前職源泉徴収票が間に合わない → 自分で確定申告
- 年途中退職で再就職なし → 確定申告（還付の可能性大）
- 退職～再就職間の国民年金・国保 → 社会保険料控除として申告可

### 会社員 年間フロー

| 時期 | やること |
|---|---|
| 毎月 | 給与から源泉徴収+社保天引き（自動） |
| 11-12月 | 年末調整書類を会社に提出 |
| 12月 | 年末調整で所得税精算 |
| 翌年1月 | 源泉徴収票受領 |
| 翌年2/16-3/15 | 確定申告（必要な場合のみ） |
| 翌年6月 | 住民税額決定通知 → 給与天引き開始 |

## 自営業者（フリーランス）の税務

### 開業時の手続きとタイミング

| 届出 | 期限 | 提出先 |
|------|------|--------|
| 開業届 | 事業開始から**1ヶ月以内** | 事業所管轄の税務署 (国税庁 A1-5) |
| 青色申告承認申請 | 青色申告したい年の**3/15まで**（1/16以降の新規開業は**開業2ヶ月以内**）| 同上 (国税庁 A1-8) |
| インボイス登録 | 任意。登録すると課税事業者に | 同上 |

### 青色 vs 白色

| | 青色 | 白色 |
|---|---|---|
| 特別控除 | 最大**65万円**（e-Tax+複式簿記）| なし |
| 赤字繰越 | 3年間 | 不可 |
| 帳簿 | 複式簿記（65万控除の場合）| 簡易記帳 |

**判断**: 事業規模が小さくても青色推奨。65万円控除のメリットが大きい

### 予定納税

- **対象**: 前年の申告納税額が**15万円以上** (国税庁 No.2040)
- **第1期**: 7/1～7/31（基準額の1/3）
- **第2期**: 11/1～11/30（基準額の1/3）
- **減額申請**: 所得激減なら 第1期は7/15まで、第2期は11/15まで
- 翌年の確定申告で精算

### 消費税

- 基準期間（前々年）の課税売上高 **1,000万円以下** → 免税事業者
- インボイス登録 → 強制的に課税事業者に
- **2割特例**（令和5年10月～令和8年9月）: 免税→インボイス登録した事業者は納税額=売上税額の**2割のみ** (フリーランス協会 2025)
- 簡易課税: 売上5,000万円以下で選択可。みなし仕入率で計算

### 自営業者 年間フロー

| 月 | やること |
|---|---|
| 毎月 | 帳簿記帳、経費管理 |
| 1月 | 源泉所得税(納期特例) 1/20 |
| **2/16〜3/16** | **所得税の確定申告** |
| **〜3/31** | **消費税の確定申告**（課税事業者のみ）|
| 4月 | 口座振替（所得税 4/23、消費税 4/30）|
| 6月 | 住民税(一期) |
| **7月** | **所得税予定納税(一期)** 7/31 |
| 8月 | 住民税(二期)、個人事業税(一期)、消費税中間申告（該当者）|
| **11月** | **所得税予定納税(二期)** 11/30、個人事業税(二期) |
| 12月 | 決算作業 |
