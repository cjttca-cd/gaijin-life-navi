# 投資（NISA / iDeCo / 証券口座）— 経験則・判断ロジック

> Agent 専用。制度の一般説明は書かない。
> 銘柄推薦・リターン予測は絶対にしない（金融商品取引法）。

## 前提ヒアリング（必須）

1. **滞在予定年数？** → 短期なら NISA/iDeCo の長期拘束がデメリット
2. **米国市民/GC？** → PFIC 問題で NISA 実質不利 → 税務アドバイザー必須
3. **マイナンバーカード？** → 証券口座開設にほぼ必須

## 証券口座開設 — 外国人の実際の障壁

### SBI 証券
- **外国人はオンライン開設不可** → 郵送のみ（7 日+）。eKYC は在留カード非対応で頻繁失敗 → **最初から郵送選択** (Reddit 2024)
- SBI Shinsei 口座がある場合、紙本申請でスムーズ
- 名前が銀行口座と完全一致していないと失敗

### 楽天証券
- 外国人は**郵送のみ**。国籍欄に「日本」のみ表示の UI バグ → 紙本に切替 (Reddit 2023)

### moomoo 証券
- 中文 UI、24h 中文客服、佣金最安級。**ただし NISA / iDeCo 非対応**（2025 確認）(XHS 2025)
- 用途: 美株取引、看盤ツール

## 券商選択ロジック

- **NISA/iDeCo 必要** → SBI or 楽天の二択
  - SBI: 商品最全。UI 古い、メンテ頻繁。米株手数料・転倉費用は 検索結果を参照（検索結果がない場合は「最新情報はウェブで直接ご確認ください」と案内） (Reddit 2025)
  - 楽天: 楽天カード定投 → ポイント還元。UI やや良
  - 両社とも iDeCo 管理費無料
- **純美株** → moomoo（手数料が安い、検索結果を参照（検索結果がない場合は「最新情報はウェブで直接ご確認ください」と案内））。NISA 不可に注意
- **銀行での証券口座は非推奨**（商品少、手数料高）
- **ポイントサイト経由で開設推奨**: 公式直接より追加ポイント (XHS 2025)

## NISA — 外国人特有の落とし穴

- **名前不一致が最大の問題**: 銀行・在留カード・証券の名前が微妙に違う → 開設失敗 or 反復補件。**最初から統一**
- **eKYC は在留カード非対応の券商多い** → 紙本が安全
- 転券商に数週間。同時 2 社で NISA 不可

## iDeCo — 実操注意

- **60 歳まで引出不可が最大リスク**: 離日しても鎖定。帰国可能性ある人は慎重に
- 口座管理費: SBI/楽天 = 無料、Resona = 3,000 円/月 → 定期 検索結果を参照（検索結果がない場合は「最新情報はウェブで直接ご確認ください」と案内）
- **自動移管リスク**: DC 退職後に手動で iDeCo 移管しないと管理費だけ引かれ投資不可状態に
- iDeCo 内の再平衡は非課税（一般口座と異なる）。反映に 1-2 週間
