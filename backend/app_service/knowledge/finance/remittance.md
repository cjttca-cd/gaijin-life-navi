# 送金 — 経験則・判断ロジック

> Agent 専用。手数料・レート等の具体的数字は web_search で最新を取得。

## チャネル選択の判断ロジック

- **日常小〜中額**: Wise が最速（数分着金）
- **大額→日本**: MUFG 口座へ（質問なし翌日着金）。**SMBC 回避**（盤問→退回リスク）
- **大額→海外**: Sony Bank / SBI 新生（高ランクで SWIFT 手数料優遇あり）
- **緊急現金**: 海外発行カード（Visa/Mastercard/銀聯）でコンビニ ATM キャッシング
- **中国向け**: Wise（支付宝直送可）/ Panda Remit（10 分着金）/ 中国銀行東京支店（中国人向け）

## Wise — 日本特有ルール

- **一類牌照取得済み**: 以前の送金上限は段階的に緩和中（最新上限は web_search で確認）(Reddit 2024)
- **残高上限あり**（全通貨合計）: 超過→凍結警告。こまめに転出（上限額は web_search で確認）(Reddit 2024)
- **日本区口座は MUFG**: 入金は国内振込扱い（振込手数料のみ）(Reddit 2025)
- 開設に **My Number Card 必須**。来日 3-4 週後に取得→申請可 (XHS 2025)
- 実体カードは平郵、届くまで約 1 ヶ月

## Wise — 風控（⚠️ 口座閉鎖リスク）

- ❌ C2C 個人間送金 / 暗号資産取引 / 長期未使用 / 手数料ゼロ追求 (XHS 2025)
- カード受取後は**大手銀行 ATM で有効化**（小銀行は吞卡リスク）
- 未使用時はカードロック、残高は Jar へ

## Wise — テクニック

- **匯率 24h ロック**: 日円ウォレット→ADD→金額入力→支払画面まで進んで放置。金曜なら月曜まで有効 (XHS 2025)
- **Auto Conversion**: 目標レート→自動換匯
- ATM 月間無料枠あり（上限額は web_search で確認）

## SMBC 海外送金の地雷

- **ネット送金廃止** → 窓口のみ。手数料が Wise 等より大幅に高い（具体額は web_search で確認）(XHS 2024)
- 窓口で全入金理由 + 送金理由 + 収款人関係を盤問。親族証明要求。手続き半日
- **口供一致問題**: 送金側「生活費」/受取側「買房」→退回+短期再送金不可。万能理由「旅行用」(XHS 2025)
- 少額送金は盤問なしの場合あり

## 海外カード ATM 取現（来日直後の命綱）

### 共通ルール
- **Visa / Mastercard / 銀聯 / AMEX** いずれもセブン ATM で取現可能
- コンビニ ATM の匯率 >> 空港両替所（大幅に有利）
- 手数料はカード種別・発行銀行で異なる → web_search で最新確認
- **来日直後は日本の口座がない** → 海外カード ATM キャッシングが唯一の現金調達手段

### Visa / Mastercard
- セブン・ファミマ・ローソンの ATM で取現可
- キャッシング枠が必要（海外出発前に有効化しておくこと）
- 一部カードは「海外キャッシング」をデフォルト無効にしている → 出発前に確認

### 銀聯（中国発行カード向け）
- セブン ATM・MUFG ATM で利用可
- 引出上限あり（1日・年間）→ web_search で確認
- ファミマは表示手数料と実際の請求額が異なるケースの報告あり (XHS 2025)
- 工商銀行カードは失敗報告あり。磁気ストライプなしは不可
- 中国側手数料免除カード → web_search で確認
