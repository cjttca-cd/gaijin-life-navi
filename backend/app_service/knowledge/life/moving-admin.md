# 引越し行政手続き — 順序と罠

> Agent 専用。一般公開情報は書かない。

## 引越し後の手続き優先順序（実戦的）

1. **転入届**（役所）— 最優先。他の全てがこれに依存
2. **国民健康保険の切替** — 転入届と同時にできることが多い
3. **マイナンバーカードの住所変更** — 転入届と同日に。90日以内にやらないとカード失効
   - ⚠️ LLM旧知識修正: 「通知カード」は **2020年5月25日に廃止済み**。現在は「個人番号通知書」が届く。個人番号通知書は**番号を知らせるだけの書面であり、身分証明やマイナンバー証明としては一切使用不可**
   - ⚠️ LLM旧知識修正: マイナンバーカードの取得は現状任意だが、**2026年6月14日から在留カードとマイナンバーカードの一体化（特定在留カード）が開始**。一体化カードを取得すれば入管での手続き後に市区町村窓口に行く必要がなくなる。取得は義務ではないが、外国人にとっての利便性が大幅に向上するため、実質的に取得が強く推奨される方向
4. **在留カードの住所変更** — 転入届提出時に裏面に自動記載（手続き不要）
5. **勤務先への届出** — 社会保険・住民税の関係
6. **運転免許証の住所変更** — 警察署 or 免許センター
7. **郵便の転送届** — 詳細は `daily-life-hacks.md` 参照。**引越し前に出せる**

## 見落としがちな罠

- 転入届を出さない → 住民票取れない → 銀行住所変更できない → 証明書届かない、の連鎖
- **同一市区町村内の引越し**: 転居届のみ（転出届不要）。知らずに二度手間になる人が多い
- **マイナポータルで転出届をオンライン提出可能** → 旧住所の役所に行かなくて済む。ただし転入届は窓口必須
- **国民年金の住所変更**: 会社員（厚生年金）は自動。自営業・無職は自分で届出必要
- **児童手当**: 転出元で「受給事由消滅届」、転入先で「認定請求書」。**15日以内**に手続きしないと支給途切れ
- **印鑑登録は転出で自動廃止** → 新住所で必要なら再登録

## 同日処理のコツ

- 役所で処理できるものを全てまとめる: 転入届 + 健保 + 年金 + マイナンバー + 印鑑登録
- **火〜木の午前中**が最も空いている。月曜・金曜・月初は混む
- 一部自治体にワンストップ窓口（一括受付）がある
