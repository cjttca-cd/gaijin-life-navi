## 3. コスト構造の詳細

### 3.1 AI API コスト推定

| ユーザー行動 | トークン/回 | 回数/DAU/日 | コスト計算 |
|-------------|-----------|------------|-----------|
| AI チャット | 2,000 tok | 3回 | 6,000 tok/DAU/日 |
| Doc Scanner | 1,500 tok | 0.5回 | 750 tok/DAU/日 |
| 合計 | | | 6,750 tok/DAU/日 |

```
Year 1 平均:
DAU = MAU × 30% = 25,000 × 0.3 = 7,500
月間トークン = 7,500 × 6,750 × 30 = 1.52B tokens
月間コスト = 1.52B × $1/1M × ¥150 = ¥228,000 → 安全マージン込みで ¥150K/月

Year 3:
DAU = 180,000 × 0.3 = 54,000
月間トークン = 54,000 × 6,750 × 30 = 10.9B tokens
月間コスト ≒ ¥1,200K/月
```

**注**: Claude API の料金低下トレンド（年率30-50%下落）を考慮すると、実際のコストはさらに低くなる可能性が高い。
